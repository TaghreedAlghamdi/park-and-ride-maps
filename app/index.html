Park and Ride Directions

Enter in an origin and destination and this widget will attempt to give you a set of directions that has the two legs, the first driving directions and the second leg public transportation.

In the latest update to this project, you can now specify your own park and ride stop (by name or address), to help alleviate holes in transit data. In another update, I hope to expand this feature so that it allows you to contribute station data so that you and others can then find it easily.

If you find any issues or would like to contribute, the project is on Github.
